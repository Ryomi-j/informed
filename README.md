## 🚀 시작하기

1. 저장소 클론

```bash
git clone [저장소-URL]
```

2. 의존성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```

4. 프로덕션 빌드

```bash
npm run build
```

## 📝 스크립트

- `dev`: Turbopack을 사용한 개발 서버 실행
- `build`: 프로덕션 빌드 생성
- `start`: 프로덕션 서버 시작
- `lint`: 코드 품질을 위한 ESLint 실행
